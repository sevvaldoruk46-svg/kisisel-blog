# kisisel-blog
HTML ve CSS kullanarak yapmış olduğum kişisel bloğuma hoşgeldiniz.



Bu projede HTML, CSS ve JavaScript kullanarak dinamik bir kişisel blog arayüzü geliştirdim. Çalışmanın temel amacı, statik bir web sayfasını JavaScript yardımıyla etkileşimli hale getirmekti.

Arayüz tasarımında HTML ile sayfanın yapısal iskeletini oluşturdum, CSS ile görsel düzenlemeleri yaparak modern ve kullanıcı dostu bir görünüm elde ettim. JavaScript tarafında ise sayfanın dinamik davranışlarını yönettim.
